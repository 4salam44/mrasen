## 🚨 إجراءات مهمة قبل الرفع

### 1. إبطال GitHub Token المكشوف (ضروري جداً)

كان هناك GitHub Personal Access Token مكشوف في الكود القديم ويجب إبطاله فوراً.

**الخطوات:**
1. اذهب إلى: https://github.com/settings/tokens
2. ابحث عن أي tokens قديمة أو مشبوهة
3. احذف أي token لا تستخدمه حالياً
4. إذا كنت تحتاج token جديد، أنشئه بصلاحيات محدودة فقط
1. اذهب إلى: https://github.com/settings/tokens
2. ابحث عن الـ token في القائمة
3. اضغط على "Delete" لحذفه
4. إذا كنت تحتاجه، أنشئ token جديد

### 2. تدوير مفاتيح Supabase (اختياري لكن موصى به)

المفاتيح التالية كانت مكشوفة في Git:
- `EXPO_PUBLIC_SUPABASE_URL`
- `EXPO_PUBLIC_SUPABASE_ANON_KEY`

**الخطوات:**
1. اذهب إلى لوحة تحكم Supabase
2. Project Settings > API
3. قم بتدوير الـ `anon` key
4. حدّث ملف `.env` المحلي بالمفتاح الجديد
